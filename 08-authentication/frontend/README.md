# Frontend

## Commands Log

```powershell
# Initialize the app project
npx create-react-app front-end --use-npm --template typescript

# Install dependencies to handle routes
npm i react-router-dom
npm i --save-dev @types/react-router-dom

# Install dependency to manage app state (data)
# Install Redux Toolkit and React-Redux
npm i @reduxjs/toolkit react-redux
npm i --save-dev @types/react-redux

# Install dependency to send HTTP requests
npm i axios

# Install dependency to handle form
npm i final-form react-final-form

# Install SASS
npm i sass
```
